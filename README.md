# Identity Runbook

This repository contains all documentation and projects for building and managing a secure identity environment. It includes step-by-step guides, screenshots, and reflections on lessons learned.  

---

## Repository Overview

This repo is organized into the following sections:

- **Environment Setup** – Initial setup and configuration steps for the lab environment.  
- **Identity and Access** – Documentation for all identity and access projects.  
- **Security Controls** – Security policies, controls, and configurations implemented.  
- **Tickets Worked** – Detailed logs of tickets handled during exercises.  
- **Runbook Projects** – Step-by-step guides for the 10 runbook projects.  
- **Screenshots** – Images and diagrams referenced in projects.  
- **Lessons Learned** – Reflections on challenges, solutions, and takeaways.
